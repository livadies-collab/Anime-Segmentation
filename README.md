# 🪄 Anime Live2D Auto-Slicer (AI Layer Separation)

Автоматизированный Machine Learning пайплайн для разделения плоских 2D-иллюстраций (аниме-персонажей) на независимые слои с последующей регенерацией фона и базовой CSS-анимацией. 

Проект создан для инди-разработчиков, VTuber-ов и 2D-аниматоров, чтобы сократить часы ручной работы в Photoshop при подготовке артов к риггингу (Live2D / Spine).

## 🚀 Что делает этот скрипт?
1. **Идеальное вырезание (Alpha Matting):** С помощью `U-2-Net` (`rembg`) персонаж попиксельно отделяется от оригинального фона.
2. **Регенерация фона (Inpainting):** `Stable Diffusion Inpainting` анализирует пустоты, оставшиеся после вырезания персонажа, и бесшовно дорисовывает задний фон.
3. **Сегментация деталей (Jigsaw Slicer):** `Meta Segment Anything (SAM)` сканирует силуэт персонажа и автоматически отделяет крупные подвижные детали (крылья, плащи, волосы) от базового тела.
4. **Анимация (Live Preview):** Код автоматически собирает вырезанные слои и применяет к ним CSS-анимацию (дыхание, взмахи) прямо в выводе Jupyter Notebook.

## 🛠 Технологический стек
*   **Computer Vision:** `rembg` (U-2-Net), `Meta SAM` (Segment Anything)
*   **Generative AI:** `diffusers`, `runwayml/stable-diffusion-inpainting`
*   **Frameworks:** `PyTorch`, `OpenCV`, `Pillow`
*   **Environment:** Kaggle / Google Colab (Оптимизировано под 1x NVIDIA T4 16GB VRAM)

## 💻 Как использовать (Быстрый старт)
Весь пайплайн собран в одном Jupyter Notebook файле.
1. Скачайте файл `notebook.ipynb` из этого репозитория.
2. Загрузите его в Kaggle или Google Colab.
3. Включите аппаратный ускоритель **GPU (T4)**.
4. Загрузите вашу картинку и укажите путь к ней в переменной `img_path`.
5. Запустите все ячейки по очереди. На выходе вы получите сохраненные PNG-слои на жестком диске виртуалки.
